# Local file storage (`FILE_STORAGE_DIR`)

Default blob root when `FILE_STORAGE_BACKEND=local` (see `.env.example`).

**Tracked in git:** demo farm layout background for Today canvas (`farm-1/layout-background/`). Seeded via `master_seed.sql` → `farms.meta_data.layout_background_attachment_id`.

**Not tracked:** receipts, zone photos, and other uploads you create at runtime — those stay on your machine unless you back them up (`make backup`).
