## Issues closed in Phase 158

| ID | Priority | Issue | Fix |
|----|----------|-------|-----|
| A1 | P0 | Citation deep links had no `aria-label` | `citationLinkAriaLabel()` + `:aria-label` on `chat-citation-link` |
| A2 | P0 | Accuracy banner not announced | `role="alert"` on `chat-accuracy-banner` |
| A3 | P0 | Chat textarea label not associated | `for`/`id` + `aria-describedby` Enter hint |
| A4 | P0 | Guardian drawer: dialog without focus trap | `useDialogFocusTrap` + Escape to close |
| A5 | P0 | No skip link to main content | Skip link in `App.vue`, `main#main-content` |
| A6 | P1 | Sidebar/mobile nav missing `aria-current` | `aria-current="page"` on active `RouterLink` |
| A7 | P1 | Zone Water/Light/Climate tabs not tablist | `role="tablist"` / `role="tab"` / `aria-selected` |
| A8 | P1 | Proposal Confirm not focused on arrival | Focus first confirm button after turn with proposals |
| A9 | P1 | High-risk proposal warning not announced | `role="alert"` on `guardian-proposal-high-warning` |
| A10 | P1 | Settings model save feedback silent | `aria-live` on save/error messages |
| A11 | P2 | Mic button emoji-only | `aria-label` on `chat-mic-button` |
| A12 | P2 | Mobile bottom nav touch targets | `min-h-[44px] min-w-[44px]` on mobile nav links |
| A13 | P2 | Trim/zero-chunk banners not status | `role="status"` on secondary banners |
