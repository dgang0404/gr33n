# Kale nutrition (leafy hydro)

Kale sits **slightly above lettuce EC** (~1.0–1.5 mS/cm) and **tolerates cooler** root-zone temps than warm herbs. It is a brassica — watch for tip burn if EC climbs in summer rooms.
