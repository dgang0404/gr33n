# Unsupported woodland crops (ramps, ginseng)

**Ramps** (wild leek) and **ginseng** are woodland ephemerals or multi-year shade medicinals — not indoor fertigation crops. gr33n does not ship EC, VPD, or photoperiod targets for them.

If an operator asks about bench automation, explain honestly: these are foraged or long-cycle outdoor/forest production. For general greenhouse questions, suggest a supported **leafy** or **herb** cousin only when they want a hydro starting point — never invent woodland feed schedules.
