# Terminology: JADAM vs natural farming

This note is for **docs, OpenAPI text, and UI copy** (not SQL schema names — the database module stays `gr33nnaturalfarming`).

## Guidelines

1. **JADAM** — Use as a **proper noun** when you mean the specific method, literature, or starter data that uses names like JMS, JLF, FFJ, WCA (e.g. “JADAM indoor photoperiod starter”, “JADAM Organic Farming” as a book citation in seed `reference_source` fields).

2. **Natural farming** — Use (lowercase) as the **generic English label** for the product area: fermented inputs, soil drenches, recipes, and inventory batches — especially in **module titles**, API tag descriptions, and navigation that should read well for international operators.

3. **Avoid** — Do not use **“Korean Natural Farming”** or **“KNF”** as the official name of the API module or primary UI heading. Some readers find tying a nationality to a whole practice imprecise or off-putting; JADAM is already the precise name for that lineage.

4. **Combining** — Phrases like **“JADAM-style inputs”** or **“natural farming inputs (JADAM-oriented demo)”** are fine when you need both specificity and category.
